构造方法

方法名和类名一致并且能初始化对象的方法

  1. 无参构造：没有参数
  2. 有参构造：有参数，参数是为指定的属性赋值
  3. 满参构造：给所有属性赋值

static

static静态修饰符
static属于类，不属于对象（比对象优先级高）

在一个工具类中，他的构造方法使用private修饰，这样可以使该类不能被new，因为工具类中的方法都是static的，所以不会影响方法的使用，
