1.String介绍

概述：

String类代表字符串

特点：

1.Java程序中的所有字符串字面值（如”abc”）都作为此类的实例（对象）实现
凡是带双引号的，都是String的对象
String s = “abc”
“abc”就是对象；String就是对象的数据类型；s就是对象名

2.字符串是常量，它们的值在创建之后不能更改
String s = “hello”
s+=”world” -> 会产生新对象

3.String 对象是不可变的，所以可以共享
String s1 = “abc”
String s2 = “abc”

3.String的创建

1.String() -> 利用String的无参构造创建String对象
2.String(String origina1)->根据字符串创建String对象
3.String(char[] value)-> 根据char数组创建String对象
4.String(byte[] bytes)-> 通过使用平台的默认字符集解码指定的 byte 数组，构造一个新的 string

对上方第五种构造方法的拓展
将char或byte数组中的其中一部分内容进行构造赋值给String对象

4.字符串常见问题

1.字符串拼接,如果等号右边是字符串字面值拼接,不会产生新对象
2.字符串拼接,如果等号右边有变量参数拼接,会产生新字符串对象

5.String的方法

boolean equals(String s) -> 比较字符串内容
boolean equalsIgnoreCase(String s) -> 比较字符串内容，忽略大小写

例：已知用户名和密码，请用程序实现模拟用户登录。总共给三次机会，登录成功与否，给出相应的提示

小技巧

1.s.equals（”abc”）中s必不能是空，不然出现空指针异常,因此，可以这样写：
“abc”.equals(s),这样就不会出现空指针异常，只会出现false了

2.当两个都是变量的对象比较时：
可以使用Objects中的equals方法，可以防空指针，即使两个都是空指针也没关系
