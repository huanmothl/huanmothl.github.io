1.Object中的toString

1.概述

Object中的toString方法：返回该对象的字符串表示形式

2.注意

  • 如果没有重写则调用Object中的toString方法，输出的是地址值
  • 可以使用一键重写直接给他生成一个
  • 当只想输出属性名不想输出地址值的时候就重写toString方法

2.Object中的equals方法

1.概述

比较两个对象的地址值是否相等

2.注意

  • 和上面的toString方法类似，也是有重写和没有重写的区别
  • 重写的分四种情况：
    1.传进来的对象是不是空对象
    2.这个比较的对象是不是他自己
    3.这个传进来的对象和他是不是一个类，不是就直接false
    4.一个类但是需要强转
