Calendar日历类介绍

Date类中的大部分对象已被弃用，现在基本上用的是Calendar日历类，

Calendar日历类用法

字段值 含义
YEAR 年
MONTH 月（从0开始，可以+1使用）
DAY_OF_MONTH 月中的天（几号）
HOUR 时（12小时制）
HOUR_OF_DAY 时（24小时制）
MINUTE 分
SECOND 秒
DAY_OF_WEEK 周中的天（周几，周日为1，可以-1使用）

练习

键盘录入一个年份,判断这一年是闰年,还是平年
能被4整除但是不能被100整除的就是闰年，其实可以直接根据这个方法进行判断，不用非得像下边这个这么麻烦，但这个还是提供了一种思路
可以将日期设置为3月1日，这样在日期上减1天就变成了2月的最后一天
