final修饰符

1.格式：

public final class 类名{}

2.特点：

被final修饰的类不能被继承

修饰方法

1.格式：

修饰符 final 返回值类型 方法名（形参）{
方法体
return 结果
}

2.特点：

被final修饰的方法，不能被重写

final和abstract不能同时使用
被final修饰的成员变量不能被二次赋值
使用final修饰对象：
1.对象中的成员变量可以改变
2.对象的地址值不能改变，即p1 = new Person("大炮",18)这一语句报错

final修饰成员变量

1.格式：

final 数据类型 变量名 = 值

2.特点：

a.需要手动赋值
final int qwe;这样会报错
b.不能二次赋值
