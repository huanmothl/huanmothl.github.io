final修饰符

1.格式：

public final class 类名{}

2.特点：

被final修饰的类不能被继承

修饰方法

1.格式：

修饰符 final 返回值类型 方法名（形参）{
方法体
return 结果
}

2.特点：

被final修饰的方法，不能被重写

final和abstract不能同时使用
被final修饰的成员变量不能被二次赋值

1.对象中的成员变量可以改变

继承

1.子类可以继承父类中私有和非私有成员，但是不能使用父类中的私有成员
2.构造方法不能继承
静态方法可以被继承使用但是不能被重写
![[Pasted image 20240720184811.png]]

继承的用法

在子类中添加extends修饰即为继承

成员变量和成员方法的访问特点

父类不能使用子类自己的成员（即父类没有子类有的成员）
子类可以使用父类自己的成员

==等号左边是谁，先调用谁中的成员变量，子类没有，找父类==

==等号右边是谁，先调用谁中的成员方法，子类没有，找父类==

成员变量看左边，成员方法看右边

方法的重写

![[Pasted image 20240720195310.png]]
就是在子类里面写一个和父类相同名称和参数列表的方法

子类中重写的方法的权限必须大于等于父类中的被重写的方法的权限
权限大小排序：
public > protected > 默认 > private
私有方法，构造方法，静态方法不能被重写

==！！注意==
子类中如果要重写父类中的一个返回值为对象的方法，
那么子类中的返回值的对象一定要是父类中的子类，
比如
当Employee中返回Employee对象时，Teacher可以返回Employee对象或Teacher对象
当Employee中返回Teacher对象时，Teacher只能返回Teacher对象

使用场景

在进行某一个方法的重启和改造时
直接进行继承，只把升级改造的方法进行重写即可

向上转型

向下转型

类似于强转，将大类型转成小类型
表现方式：

Animail cat2 = (Cat)animail ->向下转型->int i = (int)b;
